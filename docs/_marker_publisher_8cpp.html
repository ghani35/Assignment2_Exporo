Author

Date
June 2014 Description:
it publishes visualization markers, which can be used to display various types of information in a 3D environment. These markers can be used to display things like the robot's current position, a target location, or a path that the robot is following. The marker publisher node can be configured to publish markers at a specified rate, and can be used in conjunction with visualization tools such as RViz to display the markers in real-time.
